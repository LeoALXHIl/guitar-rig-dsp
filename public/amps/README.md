# Arte do amp (plug-and-play)

Solte aqui um PNG por modelo de amp e o app usa como fundo do cabeçote.
Sem arquivo, mantém o faceplate CSS + o visualizador 3D.

## Arquivos aceitos
- `amp0.png` — modelo 800-style (JCM800)
- `amp1.png` — modelo 5150-style (EVH)

## Specs
- **PNG** (fundo transparente ideal), largura ~900px, amp de frente.
- **Design original** — evite marcas registradas (Fender, Marshall, EVH…) no site público.

## Uso
Coloque `public/amps/amp0.png` (ou `amp1.png`) e recarregue (Ctrl+Shift+R).
Troca de modelo troca a imagem automaticamente.
